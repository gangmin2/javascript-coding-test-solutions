# javascript-coding-test-solutions

### 📍 commit 규칙

**commit 분류**
* `solve`: 풀이에 성공한 새 문제 추가
* `fail`: 풀이에 실패한 새 문제 추가
* `improve`: 새로운 성공 풀이를 추가한 문제
* `docs`: README 등 문서 수정  

<br>

**commit 내용**
```
  {category}: [난이도] 문제 이름
```

<br>

**commit 예시**
```bash
  git commit -m "solve: [Level 1] 문제 이름"
```
