# [Level 0] 덧셈식 출력하기

### 문제 설명
두 정수 `a`, `b`가 주어질 때 다음과 같은 형태의 계산식을 출력하는 코드를 작성해 보세요.
```
a + b = c
```

<br>

### 제한사항
* 1 ≤ `a`, `b` ≤ 100

<br>

### 입출력 예
**입력 #1**
```
4 5
```

**출력 #1**
```
4 + 5 = 9
```
