# [Bronze Ⅴ] 나머지 - 10430

|1 초|256 MB|

<br>

### 문제
`(A+B)%C`는 `((A%C) + (B%C))%C` 와 같을까?

`(A×B)%C`는 `((A%C) × (B%C))%C` 와 같을까?

세 수 `A`, `B`, `C`가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

<br>

### 입력
첫째 줄에 `A`, `B`, `C`가 순서대로 주어진다. (2 ≤ `A`, `B`, `C` ≤ 10000)

<br>

### 출력
첫째 줄에 `(A+B)%C`, 둘째 줄에 `((A%C) + (B%C))%C`, 셋째 줄에 `(A×B)%C`, 넷째 줄에 `((A%C) × (B%C))%C`를 출력한다.

<br>

|예제 입력|예제 출력|
|---|---|
|5 8 4|1<br>1<br>0<br>0|

<br>

* 구현
* 사칙연산
