### 문제
0보다 크거나 같은 정수 `N`이 주어진다. 이때, `N!`을 출력하는 프로그램을 작성하시오.

<br>

### 입력
첫째 줄에 정수 `N`(0 ≤ `N` ≤ 12)이 주어진다.

<br>

### 출력
첫째 줄에 `N!`을 출력한다.

<br>

|예제 입력|예제 출력|
|---|---|
|10|3628800|
|0|1|

<br>

### 알고리즘 분류
* 수학
* 구현
