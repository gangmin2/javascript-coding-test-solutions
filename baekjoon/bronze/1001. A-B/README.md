### 문제
두 정수 `A`와 `B`를 입력받은 다음, `A`-`B`를 출력하는 프로그램을 작성하시오.

<br>

### 입력
첫째 줄에 `A`와 `B`가 주어진다. (0 < `A`, `B` < 10)

<br>

### 출력
첫째 줄에 `A`-`B`를 출력한다.

<br>

|예제 입력|예제 출력|
|---|---|
|3 2|1|
