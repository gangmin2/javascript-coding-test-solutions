### 문제
자연수 `N`이 주어졌을 때, 1부터 `N`까지 한 줄에 하나씩 출력하는 프로그램을 작성하시오.

<br>

### 입력
첫째 줄에 100,000보다 작거나 같은 자연수 `N`이 주어진다.

<br>

### 출력
첫째 줄부터 `N`번째 줄 까지 차례대로 출력한다.

<br>

|예제 입력|예제 출력|
|---|---|
|5|1<br>2<br>3<br>4<br>5|

<br>
